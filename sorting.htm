WELCOM Sorting Algorithm Animations
การเรียงลำดับแบบแทรก ( Insertion Sort)
      เป็นวิธิการเรียงลำดับที่ทำการเพิ่มสมาชิกใหม่ เข้าไปในเซต ที่มีสมาชิกทุกตัวเรียงลำดับอยู่แล้ว และทาให้เซตใหม่ที่ได้นี้มีสมาชิกทุกตัวเรียงลำดับดัวย
การเรียงลำดับแบบฟอง ( Bubble Sort)
      เป็นการจัดเรียงลำดับข้อมูลประเภท Internal sorting โดยใช้การเปรียบเทียบคีย์ในตำแหน่งที่อยู่ติดกันทีละคู่
การเรียงลำดับแบบเลือก ( Selection Sort)
      เป็นวิธิการเรียงลำดับที่ทำการเพิ่มสมาชิกใหม่ เข้าไปในเซต ที่มีสมาชิกทุกตัวเรียงลำดับอยู่แล้ว และทาให้เซตใหม่ที่ได้นี้มีสมาชิกทุกตัวเรียงลำดับดัวย
การจัดเรียงลำดับแบบเซลล์ ( Shell Sort)
      การจัดเรียงแบบเชลล์มิธีการคือ แต่ละรอบจะแบ่งข้อมูลออกเป็นกลุ่มย่อยๆ ซึ่งแต่ละกลุ่ม โดยในการจัดเรียงลำดับคีย์ของแต่ละกลุ่มจะใช้วิธีแบบบับเบิล หรือวิธีแบบแทรกก็ได้ จากนั้นทำซ่ำใหม่จนกระทั้งไม่มีข้อมูลที่ต้องสลับอีก
การเรียงลำดับแบบเร็ว ( Quick Sort)
      เป็นวิธิการเรียงลำดับที่ทำการเพิ่มสมาชิกใหม่ เข้าไปในเซต ที่มีสมาชิกทุกตัวเรียงลำดับอยู่แล้ว และทาให้เซตใหม่ที่ได้นี้มีสมาชิกทุกตัวเรียงลำดับดัวย
